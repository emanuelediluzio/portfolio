Un sistema avanzato per il tracciamento e l'analisi delle tecnologie emergenti utilizzando i dati di arXiv. Il progetto combina la potenza di Neo4j per la gestione dei grafi con BERT per l'analisi semantica dei testi.

## Caratteristiche

- Analisi semantica dei paper scientifici
- Grafo delle relazioni tra tecnologie
- Identificazione di trend emergenti
- Visualizzazione delle connessioni
- API REST per l'accesso ai dati

## Tecnologie Utilizzate

- Python
- Neo4j
- BERT
- PyTorch
- FastAPI
- React per la visualizzazione
