Un progetto Python per la gestione e visualizzazione del curriculum vitae. Il progetto permette di creare e mantenere un CV professionale in modo semplice ed efficace.

## Caratteristiche

- Gestione dei dati personali
- Gestione delle esperienze lavorative
- Gestione delle competenze
- Esportazione in vari formati
- Interfaccia utente intuitiva

## Tecnologie Utilizzate

- Python
- SQLite
- Tkinter per l'interfaccia grafica
